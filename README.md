# 🚀 Placement Portal

A full-stack **Placement Portal Web Application** designed to streamline the campus recruitment process for students and administrators. This platform allows students to explore job opportunities, manage profiles, and apply for placements efficiently.

---

## 🌐 Live Demo

🔗 **Live Link:** https://placement-portal-1-j8yp.onrender.com/

---

## 📌 Features

### 👨‍🎓 Student Side

* User Registration & Login
* Create & Update Profile (Academic details, skills, etc.)
* Browse Company Listings
* View Detailed Job Descriptions
* Apply for Jobs
* Track Applied Companies

### 🛠️ Admin Side

* Add / Manage Companies
* Post Job Opportunities
* View Student Applications
* Manage Placement Data

---

## 🧑‍💻 Tech Stack

* **Frontend:** HTML, CSS, Bootstrap
* **Backend:** Django
* **Database:** MySQL
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/Rishikaa07/Placement_Portal.git
cd Placement_Portal
```

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Database Setup

* Create a MySQL database
* Update database credentials in `settings.py`

### 4️⃣ Apply Migrations

```
python manage.py migrate
```

### 5️⃣ Create Superuser

```
python manage.py createsuperuser
```

### 6️⃣ Run the Server

```
python manage.py runserver
```

### 7️⃣ Open in Browser

```
http://127.0.0.1:8000/
```
